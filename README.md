# DNA Viewer

An interactive 3D DNA Viewer built using OpenGL.

## Features

- Interactive 3D DNA Double Helix
- Rotation and Zoom
- Base Pair Visualization
- Learn Mode
- Quiz Mode
- Groove Mode
- DNA Replication Animation
- Auto Spin
- Dark/Light Theme

## Technologies

- C
- OpenGL
- GLFW
- GLEW
- FreeGLUT

## Compile

```bash
gcc dna_viewer.c -o dna_viewer \
-lGL -lGLEW -lglfw -lglut -lm
